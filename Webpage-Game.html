'Save Castle From Dragons' is a very interesting game developed using Python. The main aim of the game is to save the castle from the dragons by shooting arrows at them. The sprites or characters used in the game are:

  • Player Sprite
  • Dragon sprite
  • Castle sprite
  • Arrow sprite
On opening the game, you will get a welcome page where the instructions for the player are mentioned. One should press the space-bar to start the game. With every left click of the mouse, arrows should shoot out that kills the dragons on hitting them. The is a lifeline (a rectangular bar) that denotes the life of the castle. The castle life-line reduces when dragons touch the castle. The player will lose the game if the life-bar ends or if the dragon touches player. The player will win the game if it kills most dragons without losing player/castle.

Software and packages used to develop the game : The game is completely developed using Python Programming language. Coding for the game is done in an IDE called IDLE (Integrated Development and Learning Environment). This is a very simple and easy to use IDE for anyone who wants to learn Python. The module called Pygame is uded to devlop this game. Pygame is a cross-platform set of Python modules which is used to create video games. It consists of computer graphics and sound libraries designed to be used with the Python programming language.
